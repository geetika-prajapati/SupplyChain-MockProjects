# SupplyChain-MockProjects

## Trailer Flow Optimization — Mock Project
**Author:** Geetika Prajapati

**Overview**
This repository contains a self-initiated mock implementation simulating trailer/equipment flow across fulfillment centers, delivery stations, and 3P sites. The objective is to detect imbalances, propose simple rebalancing, and demonstrate operational analysis using Excel.

**Files**
- `Trailer_Flow_Optimization.xlsx` — Dataset, pivot table, and dashboard charts.
- `Trailer_Project_Summary.pdf` — One-page project summary (objective, outcome, key learnings).

**Skills demonstrated**
- Supply chain operations & asset balancing
- Practical Excel usage: pivot tables, charts, conditional formatting
- Problem framing and outcome measurement
